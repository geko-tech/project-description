## Select Project

Go to the Project Generation section

![inspector](desktop_go_to_generation)

![inspector](desktop_generation_empty)

Select the path to the project

![inspector](desktop_choose_proj_dialog)

Wait for the download to complete; this may take 1-2 minutes. A green message will appear in the terminal: "All modules are loaded."

Click the generate button

![inspector](desktop_setup_proj_finished)

Wait for the project to complete generation (if the --no-open flag is disabled, X—Åode will open with the project after generation)

![inspector](desktop_generation_finished)

For more information on changing project generation settings, see the Project Configuration section.
